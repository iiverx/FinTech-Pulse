# نبض (Nabdh) - النموذج الأول (Rule-Based Financial Health Score)

## هيكل المشروع
```
nabdh_final/
├── nabdh_pipeline.py          <- الكود الكامل (شغّليه وخلاص)
├── data/
│   └── Nabd_12_Months_Clean01.csv   <- بياناتك (12 شهر لكل مستخدم)
├── output/                    <- يتولد تلقائيًا بعد التشغيل
│   ├── Nabdh_Final_Scored.csv       <- النتيجة لكل مستخدم
│   └── Validation_Report.csv        <- نتائج اختبار الحالات اليدوية
└── README.md                  <- هذا الملف
```

## طريقة التشغيل (3 خطوات بس)

1. تأكدي عندك Python مثبت (3.8 فأكثر)، وعندك مكتبتي pandas و numpy:
```
pip install pandas numpy
```

2. افتحي Terminal / CMD داخل مجلد `nabdh_final`

3. شغّلي:
```
python3 nabdh_pipeline.py
```

بكذا بس. الكود بيقرأ البيانات من `data/`، يحسب كل شي، ويطلع لك النتائج
بمجلد `output/`، وبيطبع لك ملخص مباشرة بالشاشة.

## لو غيّرتي اسم ملف البيانات أو مكانه
افتحي `nabdh_pipeline.py`، أول سطر فيه متغير اسمه `INPUT_PATH`،
غيّري المسار فيه لمكان ملفك الجديد.

## وش يطلع لك بالنهاية (Nabdh_Final_Scored.csv)
كل صف = مستخدم واحد، وفيه:
- `Nabdh_Score` : الدرجة النهائية من 0 إلى 100
- `Nabdh_Category` : التصنيف (مستقر / متوسط / في خطر)
- كل النسب والـ Scores التفصيلية اللي ساهمت بالحساب (شفافة وقابلة للشرح)

## المنهجية باختصار
كل تفاصيل القواعد المالية والأوزان والسقوف موثقة كتعليقات (docstrings)
جوا `nabdh_pipeline.py` نفسه - افتحيه واقرأي الجزء العلوي منه، فيه شرح
كامل للمنهجية يصلح تنسخينه مباشرة لسلايد العرض.
